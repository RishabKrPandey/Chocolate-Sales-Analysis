# 🍫 Chocolate Sales Analysis

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1S6BDLvjFhYn4vKkhp89zEe7BcEMmS7Oz)

## 📝 Overview
This project is an Exploratory Data Analysis (EDA) of a fictional chocolate sales dataset. The goal of this notebook is to uncover actionable business insights by analyzing product performance, evaluating the sales team, and visualizing regional sales distributions.

## ✨ Key Features & Analysis
* **Data Cleaning & Preprocessing:** Parsed string-based currency values into numeric formats, converted date columns into datetime objects, and engineered new features like `Price Per Box`, `Month`, and `Year`.
* **Product Performance ("Cash Cows"):** Grouped and aggregated data to identify the top revenue-generating chocolate products.
* **Sales Team Evaluation:** Analyzed the performance of individual salespeople based on total revenue and the number of boxes shipped.
* **Regional Performance:** Visualized the sales distribution across different countries using pie charts and other Matplotlib/Seaborn visualizations.

## 🛠️ Technologies Used
* **Python 3.x**
* **Pandas:** For data manipulation and aggregation.
* **NumPy:** For numerical operations.
* **Matplotlib & Seaborn:** For plotting and data visualization.
* **Kagglehub:** For direct dataset downloading.

## 📂 Dataset
The dataset used in this project is sourced from Kaggle. It contains details regarding sales personnel, the country of sale, product names, dates, amounts, and boxes shipped.
> **Source:** [saidaminsaidaxmadov/chocolate-sales](https://www.kaggle.com/datasets/saidaminsaidaxmadov/chocolate-sales)

The easiest way to run and interact with this code is directly in Google Colab.
